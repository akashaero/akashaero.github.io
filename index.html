Fun Projects

  • Reduced-order Modeling of Fluid Flows: Creates Reduced-order models using high-fidelity computational fluid dynamics (CFD) data. These ROMs run multiple orders of magnitude faster than high-fidelity CFD simulations.
  • Discounted Cash Flow Analysis : A tool with graphical user interface for calculating fair value for publicly traded businesses (stocks). Request only 3 inputs from users and returns plethora of useful information about the fair value.
